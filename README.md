# Evaluating Machine Learning vs Rule-Based Systems for Decision-Making Under Uncertainty

This repository contains my research paper analyzing the effectiveness of Machine Learning (ML) compared to traditional rule-based systems across two real-world domains:

- Sports Analytics (AI-based alternative to the Duckworth–Lewis–Stern method)
- Flood Risk Prediction for Agricultural Advisory in India

The study demonstrates how ML models outperform static rule-based systems in dynamic and uncertain environments.
